# Genealopy
Python genealogy database crawler

## Prerequisites
- [Graphviz](https://www.graphviz.org/download/)

## Installing
```
cd C:­\source
git clone https://github.com/Simon-Charest/genealopy.git
cd genealopy
python -m venv .venv
.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## Executing
```powershell
python . -h
```
